We are pleased to announce a new release candidate for you to try out: **3.15.0-rc-1**. You can find it on the CDN [here](http://yui.yahooapis.com/3.15.0-rc-1/build/yui/yui-min.js), via [npm](https://www.npmjs.org/package/yui), and download it via the `.zip` [archive](http://yui.zenfs.com/releases/yui3/yui_3.15.0-rc-1.zip). We've also updated the [staging website](http://stage.yuilibrary.com/) to reflect the new release candidate.

This release candidate is currently being tested across all of our [target environments](http://yuilibrary.com/yui/environments/) and we'll be fixing showstopper issues. Please take this opportunity to test out the release candidate in your own test environments and [file issues](https://github.com/yui/yui3/issues/new) if you run into anything that seems to be not working the way you expect. Be sure to specify the version you are testing when you file the issue). **NOTE**: There are many changes to core elements, please use this time to try out this release candidate - every bit helps to make this an excellent release!

## What's New In This Release

To find out what's new in this release please check out the [Change History Rollup](https://github.com/yui/yui3/wiki/YUI-3.15.0-Change-History-Rollup). We'll have a full description of the changes in the final full release blog post, but you can get a good idea of what is coming by checking out the rollup.

## Additional Details

There were a total of **438** commits for this upcoming release by **20** contributors. You can check out all the details of this release candidate via the [GitHub Comparison](https://github.com/yui/yui3/compare/v3.14.1...v3.15.0-rc-1) to **3.14.1**.
