We are happy to announce a new release candidate for you to try out: 3.16.0-rc-1. You can find it on CDN [here](http://yui.yahooapis.com/3.16.0-rc-1/build/yui/yui-min.js), via [npm](https://www.npmjs.org/package/yui), and download it via the `.zip` [archive](http://yui.zenfs.com/releases/yui3/yui_3.16.0-rc-1.zip). We've also updated the [staging website](http://stage.yuilibrary.com/) with the latest documentation.

This release candidate is currently being tested across all of our [target environments](http://yuilibrary.com/yui/environments/) and weâ€™ll be fixing showstopper issues. Please take this opportunity to test out the release candidate in your own test environments and [file issues](https://github.com/yui/yui3/issues/new) if you run into anything that seems to be not working the way you expect. (Be sure to specify the version you are testing when you file the issue).

## What's New In This Release

To find out what's new in this release please check out the [Change History Rollup](https://github.com/yui/yui3/wiki/YUI-3.16.0-Change-History-Rollup). We'll have a full description of the changes in the final full release blog post, but you can get a good idea of what is coming by checking out the rollup.
