_Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, leave a comment below or email contributions to [weekly@yuilibrary.com](mailto:weekly@yuilibrary.com)._

-   This week the team is putting the finishing touches on **YUI 3.8**, due to be packaged up and put underneath the Christmas Tree on **December 11th**. For those of you who enjoy opening presents early, **[3.8.0pr2](http://yui.yahooapis.com/3.8.0pr2/build/yui/yui-min.js)** is already out on the CDN, so go take it for a spin!
    
-   Video from this week's **YUI Open Roundtable** [has been posted](http://www.youtube.com/watch?v=rKrzggbBpfY). Discussions include pull requests for [array-extras](https://github.com/yui/yui3/pull/299), [Y.Soon](https://github.com/yui/yui3/pull/304), and [Y.Deferred](https://github.com/yui/yui3/pull/241), as well as the [YUI Contributor mailing list](/yuiblog/blog/2012/12/05/announcing-the-new-yui-contributor-mailing-list/), DataTable ownership, and more.
    
-   **YUI's Responsive CSS Grid Builder** is now live at [yui.github.com/gridbuilder/](http://yui.github.com/gridbuilder/). This is a new open-source project, and you can submit pull requests and file issues @ [github.com/yui/gridbuilder](https://github.com/yui/gridbuilder).
    
-   [PhantomJS](http://phantomjs.org/) & [Esprima](http://esprima.org/) author Ariya Hidayat ([@ariyahidayat](https://twitter.com/ariyahidayat)) wrote about [Istanbul](https://github.com/yahoo/istanbul), an open-source JavaScript code coverage tool from Yahoo that is now being used within [Yogi](https://github.com/yui/yogi/). You can read his post at [JavaScript Code Coverage with Istanbul](http://ariya.ofilabs.com/2012/12/javascript-code-coverage-with-istanbul.html).
    
-   Jeff Pihach ([@fromanegg](http://www.twitter.com/fromanegg)) wrote a [nice post](http://fromanegg.com/post/37222972936/using-after-event-listeners-to-react-to-attribute) about using `after()` event listeners for reacting to attribute value changes.
    
-   New YUIConf videos this week:
    
    -   Caridy Patiño's [Mojito for YUI Developers](/yuiblog/blog/2012/12/07/yuiconf-2012-talk-caridy-patino-on-mojito-for-yui-developers/)
    -   Eric Ferrauiolo's [YUI, Open Source, and Community Town Hall](/yuiblog/blog/2012/12/05/yuiconf-2012-talk-yui-open-source-and-community-from-eric-ferrauiolo/)
    -   Anthony Pipkin's [Building Modules for the New Gallery](/yuiblog/blog/2012/12/03/yuiconf-2012-talk-anthony-pipkin-on-building-modules-for-the-new-gallery/)
-   YUI Team On the Road:
    
    -   I attended The Future of Web Apps conference in Prague to deliver a talk about the latest and greatest Web development tools Y! has to offer, including YUI, YQL, Mojito, and others. You can read more about that trip on the [YDN Blog](http://developer.yahoo.com/blogs/ydn/posts/2012/12/ydn-at-future-insights-praha/).
    -   Matt & Eric attended last week's TC-39 (ECMAScript) meetings to discuss the latest [ES6](https://wiki.mozilla.org/ES6_plans) proposals. You find find notes from that 3-day event [here](https://github.com/rwldrn/tc39-notes/blob/master/es6/2012-11/).
-   Version bumps for the following YUI dev tools: [Shifter](https://github.com/yui/shifter/), [YUIDoc](https://github.com/yui/yuidoc/), and [Yogi](https://github.com/yui/yogi/). Upgrade with `npm install -g shifter yuidocjs yogi`.
    
-   Updated modules this week's [Gallery build](http://yuilibrary.com/gallery/buildtag/gallery-2012.12.05-21-01): [Alea Pseudorandom Number Generator](http://yuilibrary.com/gallery/buildtag/alea/), [Any Base Converter](http://yuilibrary.com/gallery/buildtag/any-base-converter/), [Array Iterate](http://yuilibrary.com/gallery/buildtag/array-iterate/), [Bottle - Mobile UI Library](http://yuilibrary.com/gallery/buildtag/bottle/), [Busy](http://yuilibrary.com/gallery/buildtag/busy/), [Button Plugin](http://yuilibrary.com/gallery/buildtag/button-plugin/), [Datatable Resize Plugin](http://yuilibrary.com/gallery/buildtag/datatable-col-resize/), [DataTable Paginator](http://yuilibrary.com/gallery/buildtag/datatable-paginator/), [Datatable Selection](http://yuilibrary.com/gallery/buildtag/datatable-selection/), [Flyweight Tree](http://yuilibrary.com/gallery/buildtag/flyweight-tree/), [Flyweight TreeView](http://yuilibrary.com/gallery/buildtag/fwt-treeview/), [Itsa Selectlist](http://yuilibrary.com/gallery/buildtag/itsaselectlist/), [Magnifier](http://yuilibrary.com/gallery/buildtag/magnifier/), [More Results Loader](http://yuilibrary.com/gallery/buildtag/more-loader/), [Paginator View](http://yuilibrary.com/gallery/buildtag/paginator-view/), [Primrose](http://yuilibrary.com/gallery/buildtag/primrose/), and [Treeble](http://yuilibrary.com/gallery/buildtag/treeble/). FYI, this was the first Gallery build using the new [Grifter](https://github.com/yui/grifter) dev tool that should be released in the coming weeks. Woo!
