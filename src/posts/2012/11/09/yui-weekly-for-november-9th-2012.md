_Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you'd like to share, leave a comment below or email contributions to weekly at yuilibrary.com._

-   6 days until [YUIConf 2012](http://lanyrd.com/2012/yuiconf/)! Check out our amazing lineup of speakers and [register](http://www.regonline.com/builder/site/Default.aspx?EventID=1149235) today!
    
-   This week saw the release of **YUI 3.8.0pr1**, the first preview release of the YUI's 3.8 branch. Updates and additions to 3.8 include; [Y.Template](http://stage.yuilibrary.com/yui/docs/template/), [Y.Color](http://stage.yuilibrary.com/yui/docs/color/), [Y.AttributeObservable](http://stage.yuilibrary.com/yui/docs/api/classes/AttributeObservable.html), [Y.BaseObservable](http://stage.yuilibrary.com/yui/docs/api/classes/BaseObservable.html), [ScrollView enhancements](https://github.com/yui/yui3/blob/3.x/src/scrollview/HISTORY.md#380), and more! You can read all about it in the [release announcement](/yuiblog/blog/2012/11/07/yui-3-8-0pr1-template-and-color-utilities/).
    
-   `Y.Color` has 3 new demos that are now available to begin experimenting with, including [RGB Slider](http://stage.yuilibrary.com/yui/docs/color/rgb-slider.html), [HSL Color Picker](http://stage.yuilibrary.com/yui/docs/color/hsl-picker.html), and [HSL Harmony](http://stage.yuilibrary.com/yui/docs/color/hsl-harmony.html). Fun!
    
-   Yeti just landed support for using multiple instances of the same browser to speed up browser testing. The feature is available for those [installing in-development snapshots](http://yeti.cx/quick-start/index.html#Install-latest-Yeti-snapshot%20%3Chttp://yeti.cx/quick-start/index.html#Install-latest-Yeti-snapshot) of Yeti. A release is coming soon, so watch the [Yeti blog](http://yeti.cx/blog%20%3Chttp://yeti.cx/blog) for updates coming before next week's YUIConf.
    
-   Version bumps this week for the following YUI devtools: [yuidoc](https://github.com/yui/yuidoc/) and [yuitest-coverage](https://github.com/yui/yuitest). Upgrade with `npm install -g yuidocjs yuitest-coverage`. YUIDoc now has a doc linting feature, `yuidoc --lint`.
    
-   New in the [Gallery](http://yuilibrary.com/gallery/) this week is [busy](http://yuilibrary.com/gallery/show/busy), an easy way to create "busy" overlays ([example](http://www.orangebits.nl/code/gallery-busy/demo.html)). Updated Gallery modules include [slidecheckbox](http://yuilibrary.com/gallery/show/slidecheckbox), [twitter-widget](http://yuilibrary.com/gallery/show/twitter-widget), [textarea-expander](http://yuilibrary.com/gallery/show/textarea-expander), [itsadialogbox](http://yuilibrary.com/gallery/show/itsadialogbox), and [zui-rascroll](http://yuilibrary.com/gallery/show/zui-rascroll).
