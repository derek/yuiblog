YUI Weekly for December 14th, 2012

By Derek GathrightDecember 14, 2012

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, leave a comment below or email contributions to weekly@yuilibrary.com.

  • This week saw the official release of YUI 3.8, which introduces Y.Color, new templating utilties, observability in Attribute and Base, ScrollView enhancements, and much more. Happy upgrading!

  • In "YUI Moving from Builder to Shifter", Andrew Wooldridge gives an overview of Shifter, YUI's new Node.js-based build tool.

  • In this week's YUI Open Roundtable we discussed Promises, Open Sauce, performance benchmarking, Ticket #2532677, static analysis tools, Widget Popup, and more. You can find video of that meeting here, and check out the YUI Open Roundtable wiki for more details on previous and future Roundtables. We also created an Open Roundtable playlist on our YouTube channel.

  • New YUIConf 2012 videos this week: Douglas Crockford's keynote "Monads and Gonads" and Evan Goer's "Writing for Developers"

  • Daniel Barreiro (aka: Satyam) contributed a post to describe "The Flyweight Pattern" and introduce the FlyweightTreeManager and FWTreeView Gallery modules.

  • A user on Quora asked "How does YUI 3 achieve responsive web design?"

  • In "YUI From the Inside", YUI contributor Ryan Grove (@yaypie) praises the project for introducing more transparency and ease of contribution, which he requested a few months ago in "YUI from the Outside". Thanks Ryan!

  • Luke, Tilo, and Eric got together talk about issues around the tap event and the native DOM events it's built on.

  • Version bumps this week for the following YUI devtools: grover, shifter, yogi. Upgrade with npm install -g shifter grover yogi.

  • New and Updated modules in the Gallery this week: alea, array-unnest, bottle, button-plugin, composite-image, datatable-paginator, datatable-selection, flyweight-tree, fwt-treeview, paginator-view, and weighted-list.
