YUI Weekly - Aug 24th, 2012

By Derek GathrightAugust 24, 2012

  • Early in the week Dav sent an email to the team, "I couldn't sleep last Thursday night so..." [at this point, you know something awesome happened] "... so I stayed up and wrote a builder replacement for the YUI 3 project." Introducing Shifter! We're in the process of removing all Ant remnants (build.xml files, .properties files, etc.) and officially using Shifter as YUI's build tool. So if you are a developer who builds YUI from source, npm -g install shifter. Depending on your system, you should see build times improved anywhere from 30% to 1000%! More to come.

  • As part of a new builder, we published a package called yui-lint. These are our recommended JSLint configurations for doing YUI development, and will be used on all of our development tools.

  • If you missed it, we published our YUI Target Environments matrix, which is YUI's list of testing environments. Based on traffic both inside the Y! network and across the rest of the Web, these are the browsers (and JS engines) that have a sizable enough user base that warrants testing of the library.

  • It's coming up to that time of year again, YUIConf! We're still in the early-planning stages, so nothing to announce just yet aside from sharing that we're targeting early-November. So time to start thinking about ideas for talk proposals. Stay tuned!

  • YUI's testing tool Yeti now displays code coverage results. See commit fe4a563a20 for more details.

    yeti src/widget-stdmod/tests/unit/widget-stdmod.html
Connected to http://localhost:9000
 Agent connected: Safari (6.0) / Mac OS
✔ Testing started on Safari (6.0) / Mac OS
Testing... \ 100% complete (1/1) 190.78 tests/sec 
93% line coverage ✔ Agent completed: Safari (6.0) / Mac OS
8 tests passed! (883ms)

  • If you are in an OSX/Linux/BSD environment, here's a quick alias you can run daily to stay up to date with the latest changes in our developer tools, echo "alias yuiupdate='sudo npm -g install yuidocjs selleck shifter grover yuitest yeti'" >> ~/.bashrc

  • Thanks to clickholddrag, ArnaudD, rgrove, and clarle for the pull requests this week. Special hat-tip to rgrove for his speed improvements to Y.merge, something that should help out across most of the library.

  • Jeff Pihach (aka: Hatch) recently wrote some blogs posts on his experience using YUI in Cordova, so if you are into mobile development with YUI, give them a read: Guide to using YUI with PhoneGap (Cordova) and Loading YUI from Yahoo!’s CDN for PhoneGap (Cordova).

  • Steven Olmsted (aka: solmsted) wrote a Diablo 3 Skill Calculator using the YUI App Framework and gallery-lazy-load. So if you are curious to see either of those (or both) in action, crack open Chrome/Safari dev tools and inspect around through the JS.

  • New/Updated modules in the Gallery this week: gallery-bootstrap-alert, gallery-bootstrap-dropdown, gallery-bootstrap-tabview, gallery-bootstrap-popover, gallery-bootstrap-tooltip, gallery-bootstrap-collapse, gallery-nmpjaxplus, gallery-datatable-footerview, gallery-paginator-view, gallery-zui-scrollsnapper, gallery-zui-rascroll, gallery-zui-attribute, gallery-md-model. Thanks everyone!

If you have anything interesting to share, please leave a comment below. Thanks!
