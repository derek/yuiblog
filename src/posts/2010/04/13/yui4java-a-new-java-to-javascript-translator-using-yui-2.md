Sebastián Gurin [dropped a note on the YUI Forums this morning](http://yuilibrary.com/forum/viewtopic.php?p=11355 "YUI Library :: Forums :: View topic - yui4java") inviting the community to take a look at his new [yui4java project](http://yui4java.sourceforge.net/). The project, somewhat like [Fybit's Riatrax4JS](/yuiblog/blog/2010/02/02/fybit-riatrax4js/ "Fybit Riatrax4Js: Program YUI in Java » Yahoo! User Interface Blog (YUIBlog)"), allows you to write web applications in Java while leveraging YUI 2's component library. Sebastián calls the project experimental at this stage, but it's already showing promise:

-   [complex image viewer](http://yui4java.sourceforge.net/yui/org.sgx.j2s.imageviewer.gui.ImageViewerFrame.html "org.sgx.j2s.imageviewer.gui.ImageViewerFrame"), using YUI sliders and menu (and more);
-   [YUI widget demo](http://yui4java.sourceforge.net/yui/org.sgx.j2s.yui.test.yuidemo.Main.html "org.sgx.j2s.yui.test.yuidemo.Main"), showing how to load (and the Java for loading) a variety of YUI 2 widgets
