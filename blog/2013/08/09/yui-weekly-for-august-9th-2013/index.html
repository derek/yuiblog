YUI Weekly for August 9th, 2013

By Derek GathrightAugust 9, 2013

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, feel free to leave a comment below.

  • This week marked the last full week of development before Sprint 9's pull request deadline on Aug 11th. So if you have any new features to introduce for the next release, you only have a few days left to submit your pull request. As a reminder, you can always stay up to date with the latest development schedule over on the YUI wiki.

  • At this week’s Open Roundtable, discussion topics included Button improvements, an update on Y.Promise, and y3d. You can catch a video of the Hangout over on YouTube.

  • y3d is a WebGL library built with YUI written by Bruno Farache. Neat stuff!

  • Recent threads on the contributor mailing list include a discussion about "Smart" Model Lists and a request for feedback on button improvements.

  • Pure crossed 5000 stars on GitHub this week. Hurrah!
