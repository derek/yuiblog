YUI Weekly for March 22nd, 2013

By Derek GathrightMarch 22, 2013

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, feel free to leave a comment below.

  • In this week's release news, a bug was discovered in 3.9.0's Handlebars utility so we're working to include a fix for that into a patch release as soon as possible. For details on this issue check out pull request #539. If this becomes a full 3.9.1 release and includes other fixes unrelated to Handlebars, you can have a peek the yui:v3.9.0...yui:master compare to stay on top of all other inclusions since last week's 3.9.0 release.

  • Yeti 0.2.20 was released to introduce efficiency improvements in addition to other bug fixes. Check out Yeti's HISTORY.md for more details.

  • A new version of YUI GridBuilder was published to introduce a Chrome 25 fix, aesthetic improvements, and the ability to view unminified CSS.

  • Team member Andrew Wooldridge (@triptych) sat down with Anthony Pipkin (@apipkin) for a chat about Datatable in this week's Developer Spotlight. If you have a few minutes to spare, take a look at the survey included in that blog post and give us your feedback on what can be done to improve Datatable. Thanks!

  • As discussed on yui-contrib, YUI build results are now being sent to the yui-build group instead of yui-contrib. So if those updates interest you, be sure to subscribe to that group as well.

  • This week's Roundtable (video) featured a demo of Y.Tipsy (a tooltip widget), as well as a discussion about the next patch release and SemVer.

  • New and updated Gallery modules include WidgetPointer, csslist, sm-treeview, i18n-formats, patch-390-transition-htcbutterfly, tipsy, and test-dom.

  • Version bumps this week for the following YUI devtools: Yeti, Yogi, Shifter, and YUI Doc. Upgrade with npm install -g yeti yogi shifter yuidocjs.

  • Links of the Week (thanks to JavaScript Weekly).

    • js-git
    • asm.js in Firefox Nightly
    • How I Write Modules
    • A JavaScript refresh
    • ... and more

  • And finally, if you are disappointed with Google's decision to kill Reader and are looking for a replacement, YUI's Drag & Drop example contains an RSS reader that could serve as a basis for your own custom-built solution.
