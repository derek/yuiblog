YUI Weekly for Aug 16th, 2013

By Derek GathrightAugust 16, 2013

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, feel free to leave a comment below.

  • This week saw the release of YUI 3.12.0-beta-1, the first release candidate for the upcoming YUI 3.12 due out later this month. This release includes various improvements and fixes from 23 contributors across Button, Charts, Event, ModelList, Node, TabView, Template, and Tree. Check out the release announcement for all the details. The next milestone for Sprint 9 is the Aug 20th commit freeze pull request deadline.

  • y3d is a WebGL module that is new to the Gallery this week. Check out the playground for more demos. Nice work Bruno!

  • At this week’s Open Roundtable, discussion topics included 3.12-0-beta-1, the new release branch, and a chat with Rashad Russell about his summer as a YUI intern. You can find a video of that Hangout here.

  • Recent threads on the contributor mailing list include new discussion about default parameters and continuations of earlier threads on "smart" ModelLists and Button improvements.

  • New and updated modules in this week’s Gallery build include: bulkedit, itsacheckbox, itsamodellistsyncpromise, itsamodelsyncpromise, itsanodepromise, itsawidgetrenderpromise, layout, rocket, y3d-anim, y3d-camera, y3d-color, y3d-geometry, y3d-light, y3d-math, y3d-picker-plugin, y3d-scene, y3d-shader, and y3d-texture.
