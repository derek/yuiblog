DataTable solutions/examples

Here are a few examples thrown together (usually) in response to ydn-javascript questions. This is by no means an exhaustive example set. Please look at the following resources:

  • DataTable overview page on the YAHOO Developer Network
  • Official DataTable examples (included with the YUI download)
  • DataTable API docs
  • Satyam's examples
  • Satyam's great article series on YUIBlog
  • YUIBlog articles about DataTable

And search the ydn-javascript list archives. The list community is friendly and helpful.

The examples

  • Click vs Dblclick

    Handling both a click and double click in the column headers.

  • Select all/none checkbox column head

    Adding support for a checkbox in the column head that checks or unchecks all checkboxes in that column.

  • Custom formatter to conditionally highlight cells

    Listening to the DataTable's renderEvent to assign css classes to cells to give them a certain highlight color.

  • Applying conditional row coloring

    Using a column value to affect a custom row highlight.

  • Stop link click events

    Not technically a DataTable example, but illustrates the importance of stopping the default behavior of link click events when opening a YAHOO.widget.Dialog.

  • More detail on link behavior

    Detailing link behavior and order of operations when you have a js error in link handling code. At least this example is in a DataTable :)

  • Using tfoot for summary row

    While not officially supported, it is possible to add a tfoot to the table which will not be effected by sort operations.

  • More tfootiness

    A pared down example of adding a tfoot to a table (not in a DataTable this time).

  • Dynamically filtering records

    Using a function type DataSource to add filtering to a DataTable. Certainly not the only way to accomplish the task.

  • Multiple DataTables with synchronized sorting

    Creating multiple DataTables, but displaying only one set of column headers, and making all DataTables respond to sort requests by column head click.

  • Handling complex column values

    Extracts a column value from a nested data structure in JSON data. Also shows how to use a custom column formatter to display complex column values (in this case, an array)

  • Scrolling a container to show a cell

    Using a container element with css wih4 and overflow set. Clicking on a cell that is partially obscured will cause the overflow container to scroll to show the entire cell.

  • Placing specific Paginator controls in various locations

    Manually instantiating and rendering Paginator.ui.* components from the Paginator's rendered event callback.

  • (un)select rows with a simple click

    DataTable supports multiple row selection using CTRL/Cmd+click and SHIFT+click via onEventSelectRow. This example illustrates how to support a simple model where clicking on a row without CTRL/Cmd does not clear existing row selections, and just toggles the selected state of the target row.

  • Change DataTable columns at runtime

    Reuse an existing DataTable to provide multiple views of similar data with different sets of column definitions, changing between viewing modes in response to clicking on a series of links.
