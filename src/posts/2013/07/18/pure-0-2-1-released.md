We're excited to release [Pure](http://purecss.io) 0.2.1 today! Visit the [Github Releases page](https://github.com/yui/pure/releases/) to find out what's new in this release. A full summary of changes can be found in the [project's HISTORY.md](https://github.com/yui/pure/blob/master/HISTORY.md#021-2013-07-17). You can pull in the Pure 0.2.1 into your project by adding this `<link>` to your page:

```
<link rel="stylesheet" href="http://yui.yahooapis.com/pure/0.2.1/pure-min.css">

```

Thanks to everyone for filing bugs, sending pull requests and participating in discussions leading up to this release!
