YUI Weekly for January 18th, 2013

By Derek GathrightJanuary 18, 2013

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, leave a comment below.

  • This week saw the release of 3.9.0 Preview Release 1, which focused on Charts, JSONP, Responsive Grids, iOS6, and more. You can read all the details in the 3.9.0pr1 blog post and we are scheduled for 3.9.0 GA on January 22nd. Happy testing!

  • A congratulations to Eric Ferraiuolo (@ericf) for his nomination to Reviewer status!

  • Yesterday we flipped the switch and enabled Github's Issues as an alternative bug tracker. See the blog post for more details on this decision. We'd love to hear ongoing feedback, so let us know what you think by leaving a comment on that post.

  • Thursday's Open Roundtable session (video) featured discussions on; Normalize.css as a possible successor to CSSBase, feature detection & custom builds of YUI, legacy browser updates on the YUI Target Environments matrix, iOS6 testing, and Github Issues.

  • Three additional Hangouts were conducted this week to discuss Paginator (video), as well as recent pull requests for Date Format (video) and Data Binding (video).

  • New YUIConf 2012 talks this week include Jeff Pihach's "Use YUI to Lower Project Development Costs" and Tilo Mitra's "Building Native Windows 8 Apps with YUI".

  • Updated modules in the Gallery this week: accordion-horiz-vert, algorithms, console-test, contextmenu-view, datatable-celleditor-inline, datatable-celleditor-popup, datatable-contextmenu, datatable-editable, datatable-paginator, datetime-utils, exprbuilder, makenode, primrose, querybuilder, quickedit, and storage-lite.

  • Reads of the Week (thanks to JavaScript Weekly)

    • Using Web Workers to Speed-Up Your JavaScript Applications
    • Building A Node.JS Server That Won’t Melt
    • Source Maps 101
    • ... and more

Have a great weekend!
