Firefox Right Click Bug

Bug Description

In Firefox, click event handlers registered on the document object are called when the user right-clicks on an element.

Expected Behavior

Click event handlers should only be called if the user clicks an element with the left mouse button.

Steps to Reproduce

Right click on the link below (Yahoo!), and you'll see a JavaScript alert if you are using Firefox. If you are using another other browser (IE, Opera, Webkit, etc.) you won't see an alert.
