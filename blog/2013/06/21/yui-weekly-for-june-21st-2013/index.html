YUI Weekly for June 21st, 2013

By Derek GathrightJune 21, 2013

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you’d like to share, feel free to leave a comment below.

  • This week saw the release YUI 3.11 PR1. Our team's never-ending quest to make YUI faster and faster continues, and in this preview release you'll find some pretty fundamental changes to Attribute and Base to help out performance. As noted in the 3.11PR1 release announcement, some Base perf tests demonstrate a 6x speed increase over 3.10.3, and a 15x increase over 3.9.0! Additionally, this release includes the introduction of the Paginator utility, as well as various bug fixes and other enhancements. A reminder, June 30th is the pull request deadline for the this release, so you have approximately one more week to submit anything you would like included in 3.11 (expected to ship July 16th). Happy testing!

  • As mentioned on the contributor mailing list, YUI has created two new support forums; yui-support and yui-deprecated, so please subscribe and let us know what you think!

  • Some contributors are currently working on a YUI Cheat Sheet, so if you have anything to add, feel free to edit the wiki or chime in on the yui-contrib discussion.

  • A few days ago, a ticket was filed to propose introducing Moment.js to the library, and in the discussion Ryan Grove (@yaypie) mentioned a simple trick covered in his "When Not to Use YUI" talk at YUIConf 2012. It's always worth revisiting the fact that YUI Loader is a full featured JavaScript and CSS loader, and isn't restricted to only loading in YUI modules. Here's a demonstration of Ryan's trick in action, using Loader to include Moment.js and the popular 3D drawing library Three.js.

  • yui-purecss.docpad is a PureCSS skeleton for DocPad.

  • At this week’s Open Roundtable (video), YUI summer interns Rashad (@rashadrussell) and Patrick (@patjameson) gave demonstrations of two projects they are working on: a layout builder and some experiments with video chat using Web RTC.

  • New and updated modules in this week’s Gallery build include: beacon-listener, itsadialogbox, itsamodellistsyncpromise, itsamodelsyncpromise, sm-treeview, and template-factory.

  • In the last week, YUI has seen 13 authors pushing 67 commits, 7 pull requests merged, 9 pull requests proposed, and 12 issues closed.
