-   This week saw the release of YUI 3.7.2, [a patch release](/yuiblog/blog/2012/09/24/yui-3-7-2-patch-release/) to resolve a couple bugs around CSS loading. As has been mentioned previously, YUI releases are getting faster and more granular, so don't be surprised to see more patch releases per major version. For more information on our upcoming release schedule, you can now find that on [our Github wiki](https://github.com/yui/yui3/wiki).
    
-   Speaking of patch releases, the core team's current priority is bringing full support for YUI 3.7 in Windows 8's [four different JS runtimes](https://github.com/yui/yui3/wiki/Windows-8-JavaScript-Runtimes), which includes Internet Explorer 10 and WinJS. After running the YUI codebase through our gauntlet of 6,300+ unit tests in each Win8 environment, we've only run into a [few issues](http://yuilibrary.com/projects/yui3/report/140) and are currently being addressed. Also, thanks to Alexander Shusta ([@lexander](https://github.com/lexander)) for some [pull](https://github.com/yui/yui3/pull/278) [requests](https://github.com/yui/yui3/pull/279) to help add support for IE10.
    
-   Big hat tip to Alberto Santini ([@albertosantini](https://github.com/albertosantini)) for his App Framework example [pull request](https://github.com/yui/yui3/pull/276). One of the reasons we moved all documentation and examples into the [yui3 repository](https://github.com/yui/yui3) (in `/src/{component}/docs/`) was to allow for easier community contributions for those areas. Documentation is a really helpful way to contribute to YUI if you are looking to get started. Just ping any of the ops in #yui (on Freenode) if you are looking for ideas.
    
-   Earlier today we hosted [a session of Open Hours](http://www.youtube.com/watch?v=4TdM2UE2VVI) to discuss YUIConf presentations, which is now available on [our YouTube channel](http://www.youtube.com/user/yuilibrary). So if you are thinking about submitting a [proposal](/yuiblog/blog/2012/09/19/submit-a-talk-for-yuiconf-2012/) to speak at YUIConf (due Oct 10th), check it out to see some ideas and get tips from previous YUIConf presenters.
    
-   Version bumps this week for the following YUI devtools: [Grover](https://github.com/davglass/grover/), [Yeti](https://github.com/yui/yeti/), and [YUIDoc](https://github.com/yui/yuidoc/). Upgrade with `npm -g install grover yeti yuidocjs`. More about Yeti's recent and upcoming changes can be found on the [Yeti 0.2.11 release post](http://yeti.cx/blog/2012/09/yeti-0-2-11-released/).
    
-   New/updated modules in the [Gallery](http://yuilibrary.com/gallery/) this week: [simple-accordion](http://yuilibrary.com/gallery/show/simple-accordion), [calendar-jumpnav](http://yuilibrary.com/gallery/show/calendar-jumpnav), [paginator](http://yuilibrary.com/gallery/show/paginator), [itsaselectlist](http://yuilibrary.com/gallery/show/itsaselectlist), [itsatoolbar](http://yuilibrary.com/gallery/show/itsatoolbar), [selection](http://yuilibrary.com/gallery/show/selection), [modalshim](http://yuilibrary.com/gallery/show/modalshim), [anim-native](http://yuilibrary.com/gallery/show/anim-native), [scrollintoview](http://yuilibrary.com/gallery/show/scrollintoview), [zui-rascroll](http://yuilibrary.com/gallery/show/zui-rascroll), and [zui-scrollhelper](http://yuilibrary.com/gallery/show/zui-scrollhelper).
    
-   As we draw closer to [YUIConf 2012](/yuiblog/blog/2012/09/24/yuiconf-new-dates-set/) interest in YUI grows. Check out [these guidelines](http://yuilibrary.com/yui/docs/tutorials/contribute/) if you are interested in contributing code.
    
-   Two new pages have been added to the GitHub wiki: "[Yeti and YUI](https://github.com/yui/yui3/wiki/Yeti-and-YUI)" and "[Building YUI](https://github.com/yui/yui3/wiki/Building-YUI)".
