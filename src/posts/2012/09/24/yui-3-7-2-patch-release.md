We've released a new version of YUI today: **3.7.2**. Here's the [CDN link](http://yui.yahooapis.com/3.7.2/build/yui/yui-min.js), and it's also [available to download](http://yui.zenfs.com/releases/yui3/yui_3.7.2.zip). Please update to the latest release to take advantage of the fixes listed below. **Release notes:**

-   Included a [pull request](https://github.com/yui/yui3/pull/247) from the a member of the community [trunkseye](https://github.com/trunksye) regarding an issue with [TabView](http://yuilibrary.com/yui/docs/tabview/).
    
-   This release includes a fix for a [YUI Loader issue](http://yuilibrary.com/projects/yui3/ticket/2532764). Hat tip to the YUI community for raising awareness of this issue quickly!
    
-   Also included is a fix for another issue regarding [duplicate CSS getting loaded](http://yuilibrary.com/projects/yui3/ticket/2532773). Thanks to [Wei Wang](http://yuilibrary.com/forum/memberlist.php?mode=viewprofile&u=10040) for the simplified test case that helped to resolve this issue.
    

As you can see, your input can have a significant impact! Keep those pull requests and tickets coming!
