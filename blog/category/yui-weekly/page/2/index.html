YUI Weekly for March 7th, 2014

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community.

Release News

YUI

We shipped YUI 3.15.0 last week, and started work on our next sprint. The development schedule has been posted on the YUI Wiki. Our goal is to have another release ready by April 13, 2014.

Pure

The Pure team released the Grunt and Rework tools that they have been working on over the last quarter. These tools make writing CSS more enjoyable, and are not just restricted to Pure. If you write CSS, you may find these tools handy.

You can check them out on Pure’s new Tools page. The new tools are:

  • grunt-pure-grids: Generate custom mobile-first responsive grids
  • grunt-stripmq: IE fallback for mobile-first CSS
  • grunt-css-selectors: Mutate CSS selectors

Stay tuned for more from Pure next week!

YUI Open Roundtable

Our YUI Open Roundtable this week featured a conversation with Andy Earnshaw, the author of Intl.js. Intl.js provides an important polyfill for the ECMAScript Internationalization API (ECMA-402) for both client side and server side javascript. You can also find out more about Intl support in Modown in a previous Open Roundtable.

YUIConf Videos

This week, we released the video for Reid Burke’s talk on Testing YUI Everywhere. In addition, we released the Lightning Talks video. Check out this blog post for more details on the Lightning talks.

The World of JavaScript

Here are some good articles to get you up-to-speed on what’s been happening in the world of JavaScript this week.

  • You’ve surely heard of Atom by now, right? I’m personally still a big Sublime Text fan but folks really seem to be liking Atom’s plugin infrastructure.
  • Luke Hoban has a great overview of all ES6 features over at this GitHub repo.
  • The Yeoman folks wrote a nice blog post detailing how you can use Grunt and Gulp tasks to optimize performance.

Enjoy the weekend, folks!

Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community.

Release News

Work has started on Sprint 12, which will culminate in a YUI Release on February 25th. That’s only three short weeks away!

Open Roundtable

In this week’s Open Roundtable, Alberto and Clarence introduced us to Modown. We talked about design goals, how to incorporate Modown into a project, and discussed a few of the components in greater detail. YUI Weekly for January 24, 2014

Another week has rolled by, which means another edition of YUI Weekly has made its way to your browser of choice. Let’s find out what’s happened this week in the world of YUI.

Release News

We finalized our sprint schedule this week. It’s on the YUI Wiki but here are the important dates:

  • Jan 28, 2014: Sprint Begins.
  • Feb 11, 2014: Code Freeze Pull Request Deadline.
  • Feb 14, 2014: Code Freeze.
  • Feb 18, 2014: Commit Freeze Pull Request Deadline.
  • Feb 21, 2014: Commit Freeze.
  • Feb 25, 2014: Stable Release.

Open Roundtable

In this week’s Open Roundtable, Eric and I joined Andrew, Clarence and Juan to discuss the work that we’ve been doing to make CSS easier and more enjoyable to write.

If you haven’t seen the video, you’re missing out because I live-coded a responsive layout from scratch using the new mobile-first grid system that we’ve been working on.

Here are some of the tools that we mentioned in the talk:

  • rework-pure-grids: A Rework plugin to create mobile-first responsive grids.
  • grunt-stripmq: A Grunt plugin that makes sure your mobile-first layouts still look great in old browsers that don’t support media queries.
  • css-mediaquery: A media query parser, written in JS.
  • grunt-css-selectors: A Grunt plugin that lets you mutate CSS selectors within your CSS file. This is how YUI consumes Pure, but we’ve generalized it to work with any CSS input.

Stay tuned for more information as we finalize our story around responsive web design.

YUIConf Videos

There were some more YUIConf videos out this week! Are you caught up yet?

  • Reintroducing AlloyUI by Eduardo Lundgren
  • Moonwalking with JavaScript by Markandey Singh

Other News

  • Tony Pipkin wrote a nice article on Datatable and Paginator developments so make sure to check that out if you are interested in those widgets.

Catch y’all next week!
